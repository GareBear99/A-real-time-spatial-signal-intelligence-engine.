# Roadmap

## Phase 1 — Canonical Cleanup
- Establish one canonical runtime package
- Preserve legacy source as reference
- Add README, metadata, and docs
- Prepare GitHub-ready structure

## Phase 2 — Auditability Upgrade
- De-bundle app runtime into explicit source modules
- Add deterministic scenario fixtures
- Add geometry and estimator tests
- Introduce simulation session export/import

## Phase 3 — Research Shell Upgrade
- Add configurable structure datasets
- Add reproducible signal/noise presets
- Add multi-device scenario playback
- Add calibration and replay tools

## Phase 4 — Presentation Upgrade
- Add diagrams, screenshots, and walkthrough GIFs
- Add GitHub Pages demo deployment
- Add release packaging and changelog discipline
