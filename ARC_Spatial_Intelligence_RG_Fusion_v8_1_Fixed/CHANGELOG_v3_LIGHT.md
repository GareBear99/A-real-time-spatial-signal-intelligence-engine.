# ARC Spatial Intelligence RG Fusion v3 — LuciferAI Light Upgrade

## Added / changed
- Default theme switched to light.
- LuciferAI purple replaces the old cyan-heavy identity across the command-center shell.
- Map surface now defaults to topo on light mode and keeps dark as optional operator mode.
- Cards, HUD panels, logs, and feed surfaces were lightened into a cleaner geospatial SaaS style.
- Map overlays now use purple-forward structure, estimate, and radius rendering with green preserved for truth/success states.
- Runtime text updated to LuciferAI branding.

## Notes
- The engine remains local-first and session-compatible with previous RG fusion saves.
- Dark mode is still available from Settings.
