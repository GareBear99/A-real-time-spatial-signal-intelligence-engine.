# Security Policy

## Supported status

This repository is currently a research / architecture foundation and does not yet claim production deployment status.

## Reporting a vulnerability

If you discover a security issue in repository content or future implementation code, please report it privately through GitHub security reporting if enabled, or open a private channel with the maintainer rather than posting exploit details publicly.

## Scope note

This project must not be used to facilitate unlawful surveillance, unauthorized interception, or malicious signal analysis activities.
